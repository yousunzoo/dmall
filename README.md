# 📌 동아제약 공식몰 랜딩 페이지 클론코딩

✔️ HTML, CSS 만을 활용하여 해당 홈페이지의 UI를 그대로 구현해보는 과제입니다.

<br>

## ✨ 목표

- 기존 페이지의 `div` 태그를 시멘틱 태그로 대체하기.
- 기존 페이지의 `flex`와 `grid` 속성으로 대체하기.
- `class` 이름 정할 때 BEM 방법론을 도입해보기.
- Parcel bundler와 SCSS 사용하기.
- Swiper.js로 슬라이드 기능 구현하기.
- 자바스크립트가 필요한 부분은 CSS로 대체하기. (가능한 부분만!)

## ✅ 구현한 기능

- `.mainCont__visual`, `.mainCont__product-special` 영역에 Swiper.js 사용
- `.footer__family-site`의 `li`에 `tabindex` 속성을 부여하고, 해당 `li`가 focus 되면 숨겨졌던 `.family-site-sub` 등장 및 `transition` 처리 (JS 기능 CSS로 대체)
- `.mainNav__bottom`에 `position : sticky` 적용 (JS 기능 CSS로 대체)
